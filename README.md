# React + Styled Components + Babel + Webpack
> Use as template for your project.

## Commands
### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

### Build

```bash
npm run build
```

### Structure

```ascii
project
  src
    components
      Component
        index.jsx (component declaration)
        index.styles.js (styled-components declarations)
  public
    index.html
    fonts
    images
```
